# Comparison Deck Template

## Purpose
Compare multiple competitors, vendors, options, or approaches with a structured analysis.

## Slide Structure

### 1. Title Slide
- **Type:** title
- **Content:**
  - Main title: "{Topic} Comparison"
  - Subtitle: Date and context
  - Presenter (if applicable)

### 2. Executive Summary
- **Type:** content
- **Content:**
  - 3-5 key findings from the comparison
  - Overall recommendation preview
  - Scope of comparison (what's included/excluded)

### 3. Entity Slides (Repeatable)
- **Type:** content
- **Count:** One per entity being compared
- **Content per slide:**
  - Entity name as title
  - 4-6 key characteristics or features
  - Strengths (2-3 bullets)
  - Weaknesses/gaps (2-3 bullets)
  - Notable differentiators

### 4. Comparison Table
- **Type:** table
- **Structure:**
  - Rows: Comparison criteria/features
  - Columns: Entities being compared
  - Cell content: Status (Yes/No/Partial) or brief notes
  - Highlight: Use accent color for standout features

### 5. Recommendations
- **Type:** content
- **Content:**
  - Primary recommendation with rationale
  - Alternative options and when to consider them
  - Next steps or decision timeline
  - Open questions (if any)

## Content Guidelines

- Keep entity slides balanced in length
- Use consistent criteria across all entities
- Highlight differentiators with accent colors
- Include source citations where applicable
- Focus on decision-relevant information
