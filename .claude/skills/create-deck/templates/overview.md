# Overview Deck Template

## Purpose
Provide a comprehensive summary or introduction to a topic, feature, or initiative.

## Slide Structure

### 1. Title Slide
- **Type:** title
- **Content:**
  - Main title: "{Topic} Overview"
  - Subtitle: Context or timeframe
  - Date

### 2. Key Points
- **Type:** content
- **Content:**
  - 3-5 headline takeaways
  - Each point should be self-contained
  - Order by importance or logical flow
  - This slide sets expectations for the rest of the deck

### 3. Details Slides (Repeatable)
- **Type:** content
- **Count:** One per key area (typically 2-4 slides)
- **Content per slide:**
  - Clear section title
  - 4-6 supporting bullets
  - Data points or metrics where relevant
  - Visual elements if applicable (charts, diagrams)

### 4. Summary
- **Type:** content
- **Content:**
  - Recap of main points (brief)
  - Implications or impact
  - Next steps or call to action
  - Resources or references

## Content Guidelines

- Start with "why it matters" before diving into details
- Use progressive disclosure - high level to specific
- Include concrete examples where possible
- Keep technical jargon minimal unless audience-appropriate
- End with clear next steps or takeaways
