# Status Deck Template

## Purpose
Provide a project or task status update with clear progress visibility.

## Slide Structure

### 1. Title Slide
- **Type:** title
- **Content:**
  - Main title: "{Topic} Status Update"
  - Subtitle: Reporting period (e.g., "Week of Feb 3, 2025")
  - Date

### 2. Status Overview
- **Type:** content
- **Content:**
  - Overall status indicator (On Track / At Risk / Blocked)
  - Key metrics:
    - Tasks completed vs. planned
    - Progress percentage
    - Timeline status
  - One-line summary of current state

### 3. Completed
- **Type:** content
- **Content:**
  - Items completed since last update
  - 4-8 bullets maximum
  - Include completion dates if relevant
  - Highlight significant achievements

### 4. In Progress
- **Type:** content
- **Content:**
  - Currently active items
  - Expected completion dates
  - Current status/percentage if applicable
  - Owner assignments

### 5. Blockers (Optional)
- **Type:** content
- **Content:**
  - Active blockers or risks
  - Impact assessment
  - Mitigation actions
  - Help needed / escalations

### 6. Next Steps
- **Type:** content
- **Content:**
  - Planned items for next period
  - Key milestones upcoming
  - Dependencies to watch
  - Decisions needed

## Content Guidelines

- Use consistent status indicators (colors/icons)
- Be specific about blockers and help needed
- Include owners/DRIs for accountability
- Keep historical context minimal - focus on current state
- Prioritize items by importance, not just recency
