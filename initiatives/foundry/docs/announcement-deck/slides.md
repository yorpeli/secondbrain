# The Foundry — Announcement Deck

## Design System

### Theme
- **Mode**: dark
- **Style**: glass (Payoneer brand)

### Colors
| Token | Hex | Usage |
|-------|-----|-------|
| bg-base | #1E1E28 | Charcoal Black - main background |
| primary-500 | #977DFF | Neon Purple - highlights, accent |
| accent-500 | #0033FF | Electric Blue - secondary accent |
| text-primary | #FFFFFF | Pure White - headings |
| text-secondary | #888899 | Mid Gray - secondary text |
| midnight | #002373 | Midnight Blue - gradient, depth |
| off-white | #F8F9FC | Subtle backgrounds |

### Typography
- **Display**: System sans-serif, bold, large
- **Body**: System sans-serif, regular

### Effects
- Glass cards with border-white/10
- Subtle circle/dot decorative patterns at low opacity
- Gradient glows (purple/blue)
- Smooth fade/slide animations
