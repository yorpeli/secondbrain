# AB Testing Agent — Memory

## Statistical Methodology Notes

- Two-proportion z-test: compares conversion rates between control and treatment
- Significance: p < 0.05, minimum sample 100 per variant, meaningful lift > 1%
- Normal CDF via Abramowitz & Stegun approximation (no external dependencies)
- Verdicts: treatment-wins, control-wins, no-difference, insufficient-data, too-early
